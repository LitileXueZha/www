打开一个子窗口
window.open属性控制
参数取值范围说明
alwaysLoweredyes/no指定窗口隐藏在所有窗口之下
alwaysRaisedyes/no指定窗口悬浮在所有窗口之上
dependedyes/no是否和父窗口同时关闭
directoriesyes/noNav 2和Nav 3的目录栏是否可见
height像素窗口高度
hotkeysyes/no在没有菜单栏的窗口中设置安全退出热键
innerHeight像素窗口中文档的像素高度
innerWidth像素窗口中文档的像素宽度
locationyes/no位置栏是否可见
menubaryes/no菜单栏是否可见
outerHeight像素设定窗口（包括装饰边框）的像素高度
outerWidth像素设定窗口（包括装饰边框）的像素高度
resizableyes/no窗口大小是否可调整
screenX像素窗口距屏幕左边界的像素长度
screenY像素窗口距屏幕上边界的像素长度
scrollbarsyes/no窗口是否有滚动条
titlebaryes/no窗口标题栏是否可见
toolbaryes/no窗口工具栏是否可见
width像素窗口的像素宽度
z-lookyes/no窗口被激活后是否浮在其他窗口之上

例59

禁用鼠标右键

nice 我觉得这个想法不错，利用鼠标点击时定义一个回调事件来占用焦点

            document.onmousedown = function (event) {
                if (event.button == 2)  //  判断为鼠标右键
                    alert('鼠标右键被禁用');
            };  //  禁用鼠标右键
事件的.button值
值说明
0没按键
1按左键
2按右键
3按左和右键
4按中间键
5按左和中间键
6按右和中间键
7按所有键
