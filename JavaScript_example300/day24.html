以下是CSS3的新体验

圆角边框

常见的网页特效，还有border-left-top-radius、border-left-bottom-radius..这种更精细的控制

                    border-radius: 5px;  /*  css  */

                    DOM.style.borderRadius = '5px';  //  js

图形边框

虽然说挺好看的，但是下面代码一个都不能少才能显示，醉了

                    border-image: url("pic/星空1.jpg") 20 20 stretch !important;
                    width: 200px;
                    height: 150px;
                    border: 20px solid transparent;

图片阴影

说一下啊，一二偏移量，三模糊，四缩小范围，五颜六内向*

                    box-shadow: 10px 10px 15px 5px black inset;  /*  css  */

                    DOM.style.boxShadow = '10px 10px 15px 5px black inset';  //  js

过渡效果，洋名叫hover

嘿嘿嘿嘿嘿，之前以为的只有js才能实现的，现在你试试看.——这里关系必须是父子元素，还有渐褪效果是在元素本身加上个transition

                    #d2:hover #d1{
                        background: darkred;
                        transition: 2s;
                    }

动画效果，更高大上

贼高大上，@keyframes:几个关键帧组成，0%和100%是必须的或者from和to

                    @keyframes myAnimation1 {
                        0% {
                            background: darkgreen;           | animation: myAnimation1 5s ease-in 0s 10 alternate;
                            border-width: 1px;               | 调用动画：
                        }                                    | 1. 动画名 - animation-name
                        40%{                                 | 2. 持续时间 - animation-duration
                            background: darkred;             | 3. 动画方式 - animation-timing-function
                            border-width: 1px;               | 4. 延迟 - animation-delay
                        }                                    | 5. 播放次数 - animation-iteration-count
                        100% {                               | 5. 平滑播放 - animation-direction
                            background: darkred;             | 兼容webkit浏览器(safari / chrome)：
                            border-width: 20px;              | @-webkit-keyframes{ ...
                        }
                    }

图片旋转

                    transform: rotate(30deg);  /*  css  */
